Object Oriented Programming (OOP)

(Paradigm based on objects)

class : user-defined data type - collection of instances - blueprint/template

class is keyword which is used to create class

objects : instance of class - member of class

  • Public
  • Protected
  • Private

Inheritance : parent class - child class

Encapsulation : wrapping of data

Abstraction : hiding of unnecessary proparties

Polymorphism : more than one form

In ES5 : It is called Prototype

In ES6 : It is Object Oriented Programming(OOP)

Creating a Prototype / OOP

        const employee = {
            id : 1001,
            emp_name : "Ram",
            salary : 25000,
            dpart : "Electronic",
            framework : ['Angular', 'Vue js', 'Django', 'Laravel'],
            department : function(){
                console.log(this);
                return `Employee department is : ${this.dpart}`
            }
        }
    
        console.log(employee.id);               // 1001
        console.log(employee.framework);        // (4) ['Angular', 'Vue js', 'Django', 'Laravel']
        console.log(employee.framework[0]);     // Angular
        console.log(employee.framework[1]);     // Vue js
        console.log(employee.department());     // {id: 1001, emp_name: 'Ram', salary: 25000, dpart: 'Electronic', framework: Array(4), …}
                                                    Employee department is : Electronic

Constructor : intialize the object of class

(1) The process of creating instance is performed by a special function called Constructor.

(2) Constructor is a member function of class.

Creating a object in ES6 :

// function inside a function
    class HelloWorld{
        display()
        {
            console.log("Hello World");
        }
    }
const obj = new HelloWorld();
obj.display();                      // fucntion calling in function


// use of constructor   // no need to call a function like above
    class Animal {
        constructor (name, age)
        {
            console.log(`Your name is ${name} and age is ${age}`);
        }
    }
const obj = new Animal("Ram", 25);
